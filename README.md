# Getting Started

How to run the Project:

## Linux & MacOS

Clone the Repository:
```
git clone https://github.com/lou1008/NeuralNetworkDemo.git
```
Go into the folder and start the Programm
```
cd NeuralNetworkDemo
dotnet run
```
